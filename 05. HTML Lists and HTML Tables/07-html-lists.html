Unordered List

  • An unordered list is marked with a simple bullet point by default.
  • The second item of this unordered list.
  • Now, the third.
  • And, the last!

Ordered List

  1. An ordered list is marked with numbers by default.
  2. Now, you will see number two, for this item.
  3. Now, three!

Description List

Each pair of <dt> and <dd> shows a description for each term. That is, <dd> includes "description" corresponding to a "term" included in <dt>

Here you might put a term, or what you want to describe. It is enclosed with <dt> tags!
And here, you will write a description for the term above. This area is enclosed with <dt> tags!
